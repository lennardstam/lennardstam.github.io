Tools I use.

Desktop

  • Elementary OS. Sleek, fast and distraction free operating system.
  • Typora. A truly minimal markdown editor.
  • Enpass. To manage my passwords.
  • Insomnia. Design, and test APIs like a human, not a robot.
  • Firefox. My browser of choice.

Backup & snapshot

  • Snapper. The ultimate snapshot tool for Linux
  • Btrbk. Btrbk is a backup tool for btrfs subvolumes.
  • Duplicity. Encrypted bandwidth-efficient backup using the rsync algorithm
  • OfflineIMAP. Get the emails where you need them.

Self Hosted

  • Portainer. Container management made easy.
  • Filerun. File Sync and Sharing with markdown support.
  • Syncthing. Continuous file synchronization.
  • Gitlist. An elegant and modern git repository viewer
  • PLEX. Media player.
  • Net viel. Web interface for the notmuch e-mail system.
  • Scrutiny. Hard Drive S.M.A.R.T Monitoring.
  • Photoprism. App for browsing, organizing, and sharing your photo collection.
